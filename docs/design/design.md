# 設計管理ドキュメント

## 概要
英単語と日本語訳をカード形式でリスト管理し、表示・非表示を切り替えながら単語を覚えるための学習支援アプリ。

---

## 機能一覧

| # | 機能                                       | カテゴリ | 実装状態 |
|---|------------------------------------------|---|---|
| 1 | 英単語と日本語訳を入力してカードを追加する                    | 単語管理 | 実装済み |
| 2 | 削除ボタンでカードをDBから完全削除する                     | 単語管理 | 実装済み |
| 3 | カードを右スワイプしてリストから個別に非表示にする                | 単語管理 | 実装済み |
| 4 | **全カードの英単語テキストをまとめて隠す／表示する**             | 学習UX | **実装済み(FAB)** |
| 5 | **全カードの日本語訳テキストをまとめて隠す／表示する**            | 学習UX | **実装済み(FAB)** |
| 6 | 非表示（isVisible=false）にしたカードを全てリストに復元する | 単語管理 | 未実装 |
| 7 | 入力バリデーション（空文字チェック、重複登録防止）            | 入力UX | DB・ViewModel実装済み（UI未実装） |
| 8 | 英単語・日本語訳でのインクリメンタル検索                     | 単語管理 | DAO実装済み（UI未実装） |
| 9 | 全カードをDBから一括削除する                          | 単語管理 | DAO実装済み（UI未実装） |
| 10 | **個別カードのマスク解除（タップ操作）**                    | 学習UX | **実装済み** |

## 画面設計

### 全体レイアウト
```
┌────────────────────────────┐
│ 英単語カード（タイトル）         │
├────────────────────────────┤
│ [英単語入力（Error表示あり）]   │
│ [日本語訳入力（Error表示あり）] │
│ [カードを追加]               │
├────────────────────────────┤
│ [非表示をすべて戻す] [削除する]  │
├────────────────────────────┤
│ ← スワイプで非表示（#3）        │
│ ┌──────────────────────┐   │
│ │ [   English Mask   ] │   │  ← #4 英単語マスク
│ │ [   Meaning Mask   ] │   │  ← #5 日本語訳マスク
│ └──────────────────────┘   │
│                            │
│              ┌───┐         │
│              │ ④ │         │  ← 👁️ JP (日本語を表示)
│              ├───┤         │
│              │ ③ │         │  ← 🙈 JP (日本語を隠す)
│              ├───┤         │
│              │ ② │         │  ← 👁️ EN (英単語を表示)
│              ├───┤         │
│              │ ① │         │  ← 🙈 EN (英単語を隠す)
│              ├───┤         │
│              │ ➕ │         │  ← メインFAB (展開/閉じる)
│              └───┘         │
└────────────────────────────┘
```

### スピードダイヤル型FAB（#4, #5）のUX仕様
4つの機能を1つのボタンに集約するため、メインFABをタップすると上にメニューが展開される形式を採用します。

| ボタン | アイコン案 | 挙動 |
|---|---|---|
| **メインFAB** | `visibility` または `menu` | タップで以下の4つのサブボタンを垂直に展開/格納する。 |
| **① 英単語を隠す** | `hide_source` | 全カードの英単語を一括でマスクする。 |
| **② 英単語を表示** | `spellcheck` | 全カードの英単語のマスクを解除する。 |
| **③ 日本語を隠す** | `subtitles_off` | 全カードの日本語訳を一括でマスクする。 |
| **④ 日本語を表示** | `translate` | 全カードの日本語訳のマスクを解除する。 |

### インタラクション詳細
- **個別操作優先:** 全体でマスクしていても、特定のカードのマスク部分をタップすれば、その箇所だけ一時的に表示される（以前の案を継承）。
- **メニュー自動格納:** サブボタン（①〜④）のいずれかを押下した後は、メニューは自動的に閉じる。
- **視覚的フィードバック:** メニュー展開時は背景に薄いオーバーレイ（半透明の黒など）をかけ、操作対象を明確にする。

## データ設計

### Wordエンティティ（Roomデータベース）

| フィールド | 型 | デフォルト | 説明 |
|---|---|---|---|
| id | Int | 自動採番 | 主キー |
| englishWord | String | - | 英単語（**Unique制約あり**、DBレベル + 登録時に重複チェック） |
| japaneseMeaning | String | - | 日本語訳 |
| isVisible | Boolean | true | カード自体の表示フラグ |

- DBファイル名: `app_database`
- テーブル名: `words`
- ソート順: englishWord の昇順
- スキーマバージョン: 2（v1→v2 でenglishWordにUnique制約を追加）
